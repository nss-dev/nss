NSS 3.120 release notes
===============================

`Introduction <#introduction>`__
--------------------------------

.. container::

   Network Security Services (NSS) 3.120 was released on *8 January 2026**.

`Distribution Information <#distribution_information>`__
--------------------------------------------------------

.. container::

   The HG tag is NSS_3_120_RTM. NSS 3.120 requires NSPR 4.38.2 or newer.

   NSS 3.120 source distributions are available on ftp.mozilla.org for secure HTTPS download:

   -  Source tarballs:
      https://ftp.mozilla.org/pub/mozilla.org/security/nss/releases/NSS_3_120_RTM/src/

   Other releases are available :ref:`mozilla_projects_nss_releases`.
