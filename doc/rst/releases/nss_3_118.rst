NSS 3.118 release notes
========================

`Introduction <#introduction>`__
--------------------------------

.. container::

   Network Security Services (NSS) 3.118 was released on *18 November 2025**.

`Distribution Information <#distribution_information>`__
--------------------------------------------------------

.. container::

   The HG tag is NSS_3_118_RTM. NSS 3.118 requires NSPR 4.37 or newer.

   NSS 3.118 source distributions are available on ftp.mozilla.org for secure HTTPS download:

   -  Source tarballs:
      https://ftp.mozilla.org/pub/mozilla.org/security/nss/releases/NSS_3_118_RTM/src/

   Other releases are available :ref:`mozilla_projects_nss_releases`.
