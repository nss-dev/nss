NSS 3.120.1 release notes
=========================

`Introduction <#introduction>`__
--------------------------------

.. container::

   Network Security Services (NSS) 3.120.1 was released on *11 February 2026**.

`Distribution Information <#distribution_information>`__
--------------------------------------------------------

.. container::

   The HG tag is NSS_3_120_1_RTM. NSS 3.120.1 requires NSPR 4.38.2 or newer.

   NSS 3.120.1 source distributions are available on ftp.mozilla.org for secure HTTPS download:

   -  Source tarballs:
      https://ftp.mozilla.org/pub/mozilla.org/security/nss/releases/NSS_3_120_1_RTM/src/

   Other releases are available :ref:`mozilla_projects_nss_releases`.
